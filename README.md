# 📖 The Holy Quran: English Translation (PDF)

[![License: MIT](https://img.shields.io)](https://opensource.org)
[![Format: PDF](https://img.shields.io)](#)

هذا المستودع مخصص لتوفير نسخة رقمية عالية الجودة من ترجمة معاني القرآن الكريم إلى اللغة الإنجليزية بصيغة PDF، مع مراعاة دقة النص وسهولة التصفح.

---

## 🌟 المميزات (Features)
- **نص واضح**: خطوط محسنة للقراءة الرقمية والطباعة.
- **فهرس تفاعلي**: روابط سريعة للانتقال بين السور والأجزاء.
- **دقة الترجمة**: يعتمد المشروع على ترجمات موثوقة (مثل: Saheeh International أو Abdullah Yusuf Ali).
- **ملف واحد**: نسخة كاملة مدمجة في ملف PDF واحد ليسهل تداوله.
